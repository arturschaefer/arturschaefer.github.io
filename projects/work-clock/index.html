Tailored for Productivity

Everything you need to track shifts, manage time, and understand your work patterns in one app.

Precise Time Logging

Clock in, start breaks, and clock out with simple actions. Track custom shifts, overnight runs, and input historical data manually if you forget.

Monthly & Weekly Goals

Set custom target hours per day, week, or month. Keep tabs on your progress rings that motivate you to stay consistent.

Privacy-First

Your data belongs to you. Save logs locally in a secure Room DB. Enable Firebase integration for cross-device authentication and cloud backups only if you want to.

Material 3 UI

A light and dark mode interface built with Jetpack Compose, using Material 3 colors and native Android patterns.

Overtime & Earnings

Configure hourly wages, custom overtime rates, and tax calculations to automatically compute your estimated earnings on every payroll period.

Automatic Sync

Built-in integration with Firebase Firestore triggers automatic real-time sync when a network connection is available, preserving battery life.

WorkClock

Designed and built for developers and professionals who value tracking their time without sacrificing privacy.
